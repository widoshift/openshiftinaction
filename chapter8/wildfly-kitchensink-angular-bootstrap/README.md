========================
Level: Intermediate

What is it?
-----------
The `kitchensink-angularjs-bootstrap` quickstart demonstrates AngularJS and Twitter Bootstrap with JAX-RS, CDI 1.0, EJB 3.1, JPA 2.0 and Bean Validation 1.0.
